
# Here we can use classes like 
1. Account 
2. Bank
3. Driver

--------------

1. Account 
- This class can have multiple private variables like - 

Account holder name, Account number, contact number, account holder's address, branch, account balance
and Pin.

and have the getters and setters for each to update and get the values.

2. Bank 

- This class contains the private variables like 

Bank name, IFSC and also have the *Account* 

- And have the multiple behaviours like 

User can create account in bank, delete the account, see the account details, update the details,
withdraw money, deposit money and can check the Balance.

3. Driver
 
This class have the main method from which we can run the application.
